使用 Hexo 搭配 Github pages 做一個 Blog

發表於 2016-07-17

Hexo

Hexo 是基於 Node.js 網誌框架，文章的寫作只要使用 Markdown 語法，Hexo 就能解析轉成靜態頁面，且有許有多主題套件可以做更換。

安裝

安裝 Hexo 前，需有以下軟體：

  • Node.js
  • Git

有上述軟體時，使用 NPM 安裝 Hexo

Hexo 安裝完成後，使用 Hexo 生成 blog
